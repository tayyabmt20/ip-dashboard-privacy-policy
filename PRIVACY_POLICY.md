# Privacy Policy — IP Dashboard & System Monitor

**Last updated:** July 17, 2026

## Data We Collect

This extension collects the following information **locally in your browser**:

- **Public IP Address**: Fetched via api.ipify.org to display your network address
- **Network Location**: Fetched via ipwho.is to display city/country/ISP information
- **Browser Info**: Platform, language, screen resolution, extension version, tab count, battery level (via Chrome APIs)
- **Current Page**: URL and title of the active tab (visible only in the popup)

## How We Use Data

- All data is displayed **only inside the extension popup**
- **No data is transmitted to our servers**
- **No data is stored permanently**
- **No data is sold or shared with third parties**

## Third-Party Services

- **api.ipify.org**: Used to determine your public IP address
- **ipwho.is**: Used to determine geographic location from your IP
- **cloudflare-dns.com**: Used for DNS lookup functionality
